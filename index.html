My Excel Learning Journey

Documenting the key projects that helped me grow my Excel and VBA skills.

Project 1: Building a Data Matrix

Overview: In this project, I created a matrix with three columns: Person Name, Amount, and Employee ID. This helped me Insert data efficiently. I've came across ha situation where I had to analyze hundreds of rows of data where I had to change certain hourly rate values for only a certain employee. The problem I faced was that the data wasnt organized, Meaning there was a lot of rows where the data did not correlate with other rows. Not only did I have to analyze and make sure I wasnt missing any I had to do this across 10+ worksheets.

Steps:

  • Created a structured data table.
  • Input data for a certain person, amount, and employee ID.
  • Formatted the table for easy reading.

Key Takeaways:

I learned the importance of data organization and formatting in Excel.

Project 2: VBA Automation for Smart View Refresh

Overview: I developed a VBA script to refresh all worksheets connected to Smart View in a folder. This automation saved time and effort in manual refreshing.

Steps:

  • Wrote a VBA script to loop through worksheets and refresh data using Smart View commands.
  • Tested the script to ensure it worked as intended.

Key Takeaways:

This project taught me the power of VBA automation for repetitive tasks and how to interact with external applications like Smart View.

Project 3: VBA Script for Copying and Pasting Sheets

Overview: I created a VBA script to automate the process of copying and pasting an entire sheet for all Excel files in a folder.

Steps:

  • Wrote a VBA script to loop through Excel files in a folder.
  • Copied and pasted sheets into a master workbook.
  • Tested the script for correctness.

Key Takeaways:

This project enhanced my VBA skills and taught me how to work with multiple files programmatically.
